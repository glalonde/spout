# _context/

Session context and planning records for the Spout project.

## Structure

```
_context/
  README.md          ← this file
  plans/
    active/          ← one file per in-progress initiative (YYYY-MM-DD-<slug>.md)
    archive/         ← completed plans moved here when done
```

## Plans

Each plan file lives in `active/` while work is in progress and moves to `archive/` when complete. Git history preserves the evolution. File naming: `YYYY-MM-DD-<slug>.md`.
